# 🌟 Mystical Insights - 神秘的洞察

> カバラ数秘術×易経 統合占いプラットフォーム

古代の叡智と現代のテクノロジーを融合し、人々の自己理解と成長を支援する世界最高峰の占いプラットフォーム。

## ✨ 特徴

### 🔮 カバラ数秘術
- **ライフパス計算** - あなたの人生の目的を明らかに
- **名前分析** - エクスプレッション、ソウルナンバー、パーソナリティナンバー
- **マスターナンバー対応** - 11, 22, 33の特別な数値
- **詳細な解釈** - 各数字の意味、強み、課題、アドバイス
- **生命の樹ビジュアライゼーション** - セフィロトの木のアニメーション表示

### ⚡ 易経占い
- **64卦完全データベース** - すべての卦の詳細情報
- **手動入力対応** - コイン投げの結果を手動で入力
- **直感法** - シンプルで直感的な占い
- **詳細な解釈** - 状況、アドバイス、警告、キーワード
- **八卦情報** - 上卦・下卦の詳細な属性表示
- **爻の視覚化** - 六爻の視覚的表現と変爻表示

### 🎨 UI/UX
- **神秘的なデザイン** - 宇宙をイメージした美しいインターフェース
- **滑らかなアニメーション** - Framer Motionによる洗練された動き
- **レスポンシブ対応** - あらゆるデバイスで最適な表示
- **アクセシビリティ** - 誰でも使いやすいインターフェース
- **ナビゲーション** - 太極図アイコンを使った直感的なヘッダー
- **フッター** - 情報とリンクを含む完全なフッター

## 📦 セットアップ

### 必要な環境
- Node.js 22以上
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# 本番サーバーの起動
npm start
```

### 開発サーバー
http://localhost:3000 でアプリケーションにアクセスできます。

## 🎯 主な機能

### カバラ数秘術診断
1. 生年月日と名前を入力
2. 自動的にコアナンバーを計算
   - ライフパス
   - エクスプレッション
   - ソウルナンバー
   - パーソナリティ
   - バースデー
3. 詳細な解釈と総合分析を表示

### 易経占い
1. 質問を入力
2. 占い方法を選択（硬貨投げ/直感法）
3. 卦を立てる
4. 卦の解釈とアドバイスを表示
